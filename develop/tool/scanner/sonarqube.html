SonarScanner for .NET

  • msbuild
  1. 確認是否有安裝java
  2. 環境變數是設置$path
    • 與window search 輸入env
    • 設定環境變數path
  3. 修改$install_directory/sonarqube.analsis.xml，設置url與userId,password
  4. 安裝dotnet tool install --global dotnet-sonarcanner --version x.x.x
  5. 於專案或方案檔路徑輸入
dotnet sonarscanner begin /k:"produect key" /d:sonar.login="token"
dotnet build
dotnet sonarscanner end /d:sonar.login="token"
  1. 確認專案或方案可正常建置，不然無法產生結果
