# Solve None Dependent Image


因build dockerfile error出現<none> image
刪除iamge時會遇到 image has dependent child image
有依賴其他image，而無法移除

```powershell
docker image inspect --format='{{.RepoTags}} {{.Id}} {{.Parent}}' $(docker image ls -q --filter since=<image id>)
```

[[docker-rmi]]
```powershell
docker rmi <image id>
```
