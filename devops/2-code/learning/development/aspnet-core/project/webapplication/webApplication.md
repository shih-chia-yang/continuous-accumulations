## create web project

- create solution

```dotnetcli
dotnet new sln -o webApplication
```

- create web project

    ```dotnetcli
    dotnet new mvc -o code.web
    dotnet sln add code.web/code.web.csproj
    ```

- add a folder name `EF`, then add models
    - Person
    - Student
    - Enrolment
    - DataContext
