# Git Commit
[[git]]
確認執行提交訊息結果，但不實際執行commit，狀態為綠色準備提交中，仍需進一步輸入`git commit`
````bash
git commit --dry-run
````

可修改前一次提交訊息內容
````bash
git commit --amend
````
